# 🌐 Network Design Project

> Secure network design and configuration — coursework from IT-212 & CYB-210 at SNHU

## Overview
This repository documents two hands-on network design projects completed as part of my IT and Cybersecurity coursework at Southern New Hampshire University. Each project required designing, configuring, and documenting a secure network environment from scratch.

---

## 📁 Projects

### Project 1 — New Office Network Setup (IT-212)
**Objective:** Design a complete, secure network environment for a new office including a network, server, client systems, and firewall.

**Key concepts applied:**
- Hardware and software function in networked environments
- Secure network design principles
- Firewall placement and configuration
- Client-server architecture

**Skills demonstrated:** Network topology design, security-first thinking, technical documentation

---

### Project 2 — Network Configuration & IP Design (CYB-210)

**Part 1 — Network Configuration**
Configured network devices and implemented IP addressing to meet specified architectural requirements.

**Part 2 — Network Design with Subnetting**
Designed a full IP addressing scheme including:
- Subnet mask calculation
- DHCP server configuration
- NAT (Network Address Translation)
- PAT (Port Address Translation)
- Design rationale documentation

**Skills demonstrated:** IP addressing, subnetting, DHCP/NAT/PAT, network architecture, technical writing

---

## 📚 Courses
- **IT-212** — Intro to Computer Networks | Grade: A

## 🔑 Key Takeaways
Working through these projects gave me a hands-on understanding of how networks are designed with security in mind from the start — not bolted on afterward. Proper subnetting, firewall placement, and NAT configuration are foundational to any secure environment.
