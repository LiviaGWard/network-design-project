# 🏢 IT-212: Intro to Computer Networks — New Office Setup
**Southern New Hampshire University | Grade: A | Term: 2024 C-4 (Jul – Aug)**

> End-to-end network design project for a new office location in Fayetteville, NC — covering ISP selection, OSI model application, IP addressing, LAN topology, hardware selection, and security planning.

---

## 📋 Project Overview
The final project for this course was a complete, real-world network design for a new office. Rather than working with an existing network, I built the design from scratch — starting with ISP research and working through every layer of the OSI model to produce a fully documented network plan with hardware specifications, IP addressing strategy, topology rationale, and security considerations.

## 🔗 How IT-212 Connects to CYB-210
IT-212 provided the foundational network design knowledge — topology, OSI model, IP addressing, and hardware selection — that CYB-210 built upon with hands-on Cisco Packet Tracer configuration, subnetting, VLAN segmentation, and NAT/PAT implementation. Together they represent a complete picture: design → configure → secure.
